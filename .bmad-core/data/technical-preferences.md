# User-Defined Preferred Patterns and Preferences

## Tech Stack

### Frontend Framework
- **React Native** with **Expo**
- Expo SDK for cross-platform mobile development

### Navigation
- **Expo Router** for file-based routing

### Language
- **TypeScript** for type-safe development

### Styling
- **StyleSheet** (React Native default)
- **NativeWind** (optional, for Tailwind CSS-like styling)

## Project Conventions

### Code Style
- Use functional components with TypeScript
- Follow React Native best practices
- Prefer hooks for state management

### File Structure
- Components in `/components`
- Screens/routes in `/app` (Expo Router convention)
- Shared utilities in `/utils`
- Types in `/types`

### Testing
- Jest for unit tests
- React Native Testing Library for component tests
