# Wireframes

Wireframes as **Excalidraw**. For each wireframe commit **both**:

- `<name>.excalidraw` — the editable JSON source (open at
  [excalidraw.com](https://excalidraw.com) or the VS Code Excalidraw extension).
- `<name>.svg` — an export of the same wireframe, so it renders inline in GitHub and PR
  diffs without a viewer.

Keep the `.excalidraw` and `.svg` in sync **in the same commit**. See
[`../CONTRIBUTING-DESIGN.md`](../CONTRIBUTING-DESIGN.md) for the contribution workflow.
